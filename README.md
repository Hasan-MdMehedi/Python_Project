A simple, clean, and professional To-Do List Desktop Application built using Python (Tkinter).
This project allows users to efficiently organize their daily tasks with features like task creation, updating, deletion, marking tasks as completed, sorting by criteria, and saving/loading tasks from a file.
.

Key-Features :

Add Tasks with description, priority, and due date

Edit Tasks with a dedicated modify window

Delete Tasks from the list

Mark Tasks as Completed

Sort Tasks by:

Priority

Due Date

Completion status

Data Persistence:

Save tasks to a file

Load tasks from a file

User-Friendly GUI using Tkinter

Professional Interface with organized layout.
